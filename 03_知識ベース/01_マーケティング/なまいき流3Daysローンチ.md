# なまいき流3Daysローンチ：逆算型プロモーションの極意

なまいきくんの「3Daysローンチプログラム」から抽出した、最小の労力で最大の売上を叩き出すための「喪黒流」戦略のまとめ。

## 1. 核心的コンセプト：レバレッジの集中
- **無駄の徹底排除**: 教育ポスト、告知、動画、Space、LINE配信以外の「売上に繋がらない作業」を全てカットする。
- **「楽稼ぎ」の正体**: 実情は、売上にインパクトのある「数点のレバー」にエネルギーを100%集中させる高度な戦略。
- **再現性の担保**: 属人性ではなく、感情を動かす「型（テンプレ）」に落とし込むことで、誰がやっても一定の成果が出るように仕組み化する。

## 2. ローンチ成功のための事前準備
ここが埋まると、ポストも動画もSpaceも迷わなくなります。

### 商品情報7項目テンプレ
1.  **悩み・欲求**: 読者の抱える具体的な痛み。
2.  **原因**: なぜ解決できないのか？（間違った常識の指摘）。
3.  **解決**: あなたの手法の強み。
4.  **理想の未来**: 実現する未来を5〜10個。
5.  **強み・差別化**: 競合にはない絶対的な利点。
6.  **商品内容**: 目次のざっくりした概要。
7.  **仮想敵**: 競合の弱点や古い手法を5〜10個。

## 3. 2週間のポスト設計（3フェーズ構成）
- **フェーズ1（14〜10日前）**: 興味づけと問題点の指摘。まず注目を集め、学ぶ余白を作ります。
- **フェーズ2（10〜5日前）**: 手段の絞り込みと再現性。解決策をあなたの商品側に寄せます。
- **フェーズ3（5日前〜前日）**: 教育追加・予告・リアルタイム。最後の迷いを消し、熱量を最大化。

## 4. ポスト7種類：構成テンプレと役割
合計30〜40本を目安に投下します（予告とリアルタイムを兼ねるのがコツ）。

| 種類 | 役割 | 型・文テンプレ |
| :--- | :--- | :--- |
| **興味づけ** | すごそう！ | 「たった（期間）で（結果）が出た」「数字」「秘匿感」 |
| **問題提起** | 今のままではヤバい | 「伸びない人の共通点はこれ」「悪いのはあなたじゃない、この常識のせい」 |
| **手段削除** | これしかない | 「他はキツい、だからこれ一択」「いま必要なのはこれだけ」 |
| **再現性** | 私にもできる | 「昔の私はこうだった」「同じやり方でこの人もこうなった」 |
| **投資教育** | ブレーキを外す | 「チャンスは待たない、取る」「ここで止まる人が一生変わりませんねえ」 |
| **予告** | 期待値を上げる | 「今からこれ作る」「明日（時間）にSpaceやる」「何日にこれ出す」 |
| **リアルタイム** | 感情を動かす | 「いま作成途中の画面を見せる」「仕上げの過程を実況する」 |

## 5. 直前3日の実戦運用スケジュール
- **3日前**：動画配布告知ポスト ＋ LINEで動画配布 ＋ 感想依頼（特典付き）。
- **2日前**：**教育Space** (21時〜)。目的：欲しくさせる（売り込まない）。
- **販売当日**：販売開始ポスト ＋ LINE配布 ＋ **販促Space** (21時〜)。目的：背中を押す。

## 6. 各種台本・構成テンプレ

### ローンチ動画（7ブロック構成）
1.  **「地獄」の顕在化（依存）**:
    客観的な実績（数字）で圧倒し、現状のやり方が「地獄」であることを認識させます。「あなた自身」ではなく「手法」が悪いと定義し、正しい導きを求める状態を作ります（数々のビジネスの「死」を看取ってきた経験に基づく断罪）。
2.  **悩みの列挙**: 商品情報の「悩み・欲求」をそのまま読み上げる。
3.  **問題点の指摘**: 読者のせいにせず、仮想敵や古い常識のせいにする。
4.  **解決策の紹介**: 「何をすべきか」だけ言い、「どうやるか」は隠す。
5.  **ノウハウチラ見せ**: 1〜2割の具体例を出し、有益感を与える。
6.  **理想の未来**: 簡易性、即効性、具体的な事例の提示。
7.  **商品紹介**: 販売日時、学べること、各特典、受け取り方法。

### Space台本テンプレ
- **教育Space (2日前)**: 導入(10分・背景) → 本題(原因・解決策 35分) → 告知(動画・商品の強み 15分)。
- **販促Space (当 日)**: 導入(5分・テーマ) → 本題(商品背景・全章見どころ 35分) → オファー(期間・質問 15分)。

## 7. コピペで使う2週間スケジュールの型
- **14〜10日前**: 興味づけ(1日1〜2本)、問題提起(2日に1本)。
- **10〜5日前**: 手段の絞り込み(1日1本)、再現性(2日に1本)。
- **5〜当日**: その他教育(1日1本)、予告(1日1本)、リアルタイム(随時)。
- **6日前**: LINEで動画告知 / **3日前**: LINEで動画配布 / **2日前**: 教育Space / **当日**: 販売＆販促Space。
